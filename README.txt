## Setup

### Homepage

Le front est géré via [Gulp](https://gulpjs.com/), pour la minification des JS/CSS et la compilation Sass, le tout avec sourcemaps et autoprefixes (et polyfills si besoin mais bon)

```bash
npm i
gulp watch
```
